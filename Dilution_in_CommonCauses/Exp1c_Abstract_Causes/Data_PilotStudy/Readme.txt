- The data ".txt" file in this folder contains the raw data of the pilot study. 

in this ".txt" file, the different columns mean the following: 

"sID": 				subject ID 
"Side":				Counterbalancing factor (whether the multiple-effects cause was displayed on the left or right side)
"Q_order":			Counterbalancing factor (whether subjects first gave a strength rating for the multiple	or the single-effect cause)
"Target_effect":		Counterbalacning factor (whether the target effect was E1, E2, or E3)
"multiple":			DV (subjects' ratings for the multiple-effects cause)
"single":			DV (subjects' ratings for the single-effect cause)
"Desktop_Confirmation":		Check query (subjects had to indicate that they work on a Desktop PC [1]) 
"Attention_Confirmation":	Chech query (subjectes had to indicate that they will pay attention [1])
"Justification":		Open query (subjects were asked to write short explanations for their strength ratings)
"Age":				Subjects' age in years
"Sex":				Subjects' sex (1 = male, 2 = female, 3 = non-binary)
"Technical_issues":		Open query (subjects could report any experienced technical issues)
"Duration (sek)":		The time subjects needed to complete the study (in seconds)
